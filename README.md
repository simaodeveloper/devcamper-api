# DevCamper API

> Back-end API for DevCamper application, which id a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settgins to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
